# jazzy-vintage-cube

## Technologies

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Chakra UI](https://chakra-ui.com/)

## Local Development

### Setup

```sh
git clone <repo URL>
cd jazzy-vintage-cube
yarn
```

### Running

Finally, run the following to get the application running locally:

```sh
yarn dev
```
